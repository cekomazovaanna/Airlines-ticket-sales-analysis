# Airlines-ticket-sales-analysis
Анализ продаж авиабилетов

1. Очистка данных и удаление дубликатов, приведение столбцов к нужным типам данных

2. Анализ динамики выручки и продаж в общем и по категориям: МВЛ и ВВЛ, типам билетов по возрастам, онлайн и офлайн заказам

3. Удаление выбросов

4. Проверка гипотезы о равенстве выручки по сезонам

5. Проверка гипотезы о равенстве выручки в зависимости от времени покупки билета

6. Выявление самых популярных направлений для полета в ВВЛ и МВЛ

7. Исследование корелляции выручки с другими факторами
<b> Выводы <b>
1. Поведение покупателей, категория билета по возрасту, наличию льготной программы, сезон - можно прийти к выводу, что выручка не зависит от большинства обстоятельств. Самое значимое влияние из всех факторов оказывает летний сезон - видим слабую положительную связь.
2. Проанализировав самые популярные направления международной и внутренней авиации, можно прийти к следующим выводам:

ВВЛ

ТОП-3 популярных направления ВВЛ: Сочи-Москва, Москва-Сочи, Петербург-Москва
Самые дорогие направления: Новосибирск-Сочи, Новосибирск-Москва, Москва-Новосибирск
Сочи-Москва и Москва-Сочи лидирует, благодаря большому количеству проданных билетов; Новосибирск-Москва и Москва-Новосибирск - засчет более высокой цены на билеты при мЕньших объемах продаж
МВЛ

Москва-Ош : лидирующуу направление
Худжанд-Москва самое дорогое направление из популярных. При этом, Москва-Худжанд стоит практически на 30% дешевле. Можно предположить, что через Москву клиенты могут совершать сложные перелеты с пересадками или переезжают в Москву по ВНЖ.
В среднем, внутренние направления обходятся дешевле, чем международные: 357 против 478 рублей
Выручка внутренних направлений многократно превосходит выручку внешней авиации засчет большого количества перелетов по стране: 982 полета ВВЛ против 123 МВЛ
